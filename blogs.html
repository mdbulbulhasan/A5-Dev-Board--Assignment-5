Question-2: What is the difference between innerHTML, innerText, and textContent?
The basic difference between them:
innerHTML: Gets or sets HTML content (parses HTML).
innerText: Gets or sets visible text (repects styling).
textContent: Gets or sets all text (ignores styling, faster).
Question-3: What is event delegation in the DOM?
Event delegation is a technique where a parent element handles events for its child elements using event bubbling.It improves performance by reducing event listners.
Question-4: What is event bubbling in the DOM?
Event bubbling is a phase where an event starts from the target element and propagates up to its ancestors in the DOM.
